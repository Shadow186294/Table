# Users Table (React + Vite)

Демо: https://shadow186294.github.io/Table/

## Стек
- React 18+
- TypeScript
- Vite
- Fetch API
- GitHub Pages (деплой через GitHub Actions)

## Описание
Приложение отображает таблицу пользователей из API https://dummyjson.com/docs/users  
Реализованы сортировка через HTTP-запросы, пагинация и модальное окно с деталями пользователя.

## Функционал
- Таблица пользователей: фамилия, имя, отчество, возраст, пол, телефон, email, страна, город
- Сортировка через API (3 состояния): ASC / DESC / без сортировки
- Пагинация (переключение страниц)
- Модальное окно с подробной информацией по клику на строку
- Обработка ошибок загрузки

## Установка и запуск локально
```bash
npm install
npm run dev
